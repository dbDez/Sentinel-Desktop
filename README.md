# Safety Sentinel Desktop Application

Safety Sentinel Desktop Application is a modern WPF .NET 8 solution designed to enhance personal and organizational safety. It provides real-time threat alerts, executive briefings, geofence notifications, and daily risk scores, all in a user-friendly desktop interface.

The application integrates intelligence services, sound alerts, and customizable watchlists, helping users stay informed and prepared. With support for multiple languages and platforms, Sentinel-Desktop is ideal for professionals, travelers, and anyone seeking proactive safety management.

## Features
- Real-time threat alerts
- Executive briefings
- Geofence notifications
- Daily risk scores
- Intelligence services integration
- Sound alerts
- Customizable watchlists
- Multi-language support

## Technologies
- WPF .NET 8
- C#

## Ideal For
- Professionals
- Travelers
- Organizations
- Anyone seeking proactive safety management
